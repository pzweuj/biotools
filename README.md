# BioTools

[English](./README.en.md) | 中文

一个现代化的生物信息学工具集合，提供常用的分子生物学计算和参考工具。

## 🚀 快速开始

### 环境要求
- Node.js 18.0 或更高版本
- pnpm（推荐）或 npm

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本
```bash
pnpm build
pnpm start
```

## 🛠️ 技术栈

- **框架**: Next.js 14
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI组件**: Radix UI
- **图标**: Lucide React
